# 安装与配置

1. 推荐安装方式：PC端使用清华大学tuna镜像（https://mirrors.tuna.tsinghua.edu.cn/help/anaconda/），
2. android手机使用termux（https://www.jianshu.com/p/6fd84cc74a5e）
3. 安装Python后，pip安装依赖包

    pip install -r requirements.txt
